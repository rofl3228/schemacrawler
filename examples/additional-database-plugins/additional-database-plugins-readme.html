SchemaCrawler - Additional Database Plugins

Description

SchemaCrawler allows scripting with various databases. This example shows how to download support for some additional databases.

How to Setup

  1. Make sure that java is on your PATH
  2. Start a command shell in the _downloader directory
  3. Run download.cmd plugins (or download.sh plugins on Unix) to install additional database plugins

How to Run

  1. Run SchemaCrawler against your database.
